# Orbit

A Clean Architecture solution built using .NET 10.

## Solution Structure

- **Orbit.Domain**: Enterprise entities, enums, value objects, and common structures.
- **Orbit.Application**: Interfaces, models, DTOs, services, mappings, and validators.
- **Orbit.Infrastructure**: Database configurations, contexts, repository implementations, and external integrations (Gemini API client).
- **Orbit.Web**: MVC Web UI presentation layer.
- **Orbit.API**: ASP.NET Core API presentation layer.
- **Orbit.Tests**: Unit and Integration tests.
